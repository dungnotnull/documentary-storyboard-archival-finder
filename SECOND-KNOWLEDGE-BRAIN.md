# SECOND-KNOWLEDGE-BRAIN - Documentary Storyboard & Archival Footage

## Core Concepts & Frameworks
- Documentary structure - three-act; quest/journey; observational vs. expository vs. participatory modes (Nichols' modes); thesis-driven; the story circle.
- Storyboarding & shot lists - scene breakdown, shot types (WS/MS/CU), camera movement, Ken Burns effect on stills, A-roll (interview) vs. B-roll.
- Archival research - finding footage/images in public archives; metadata, provenance.
- Rights & licensing - Public Domain, Creative Commons, royalty-free vs. rights-managed, editorial-only; fair use is fact-specific and needs legal review.
- Coverage - ensuring each narrative beat has visual support; identifying gaps.

## License-Type Taxonomy
| License | Use allowed | Attribution | ShareAlike | Commercial | Notes |
|---|---|---|---|---|---|
| Public Domain (PD) | Yes | No | No | Yes | Verify jurisdiction; U.S. federal works usually PD. |
| CC0 | Yes | No (appreciated) | No | Yes | Public-domain dedication. |
| CC BY | Yes | Required | No | Yes | Attribute creator. |
| CC BY-SA | Yes | Required | Yes | Yes | Derivatives must be SA. |
| CC BY-ND | Yes | Required | No | Yes | No derivative works. |
| CC BY-NC | Yes | Required | No | No | Non-commercial only. |
| CC BY-NC-SA | Yes | Required | Yes | No | Non-commercial + SA. |
| CC BY-NC-ND | Yes | Required | No | No | Non-commercial + no derivatives. |
| Royalty-Free (RF) | Per license | Per license | No | Usually yes | License grant, not public domain. |
| Rights-Managed (RM) | Per license | Per license | No | Per license | Restrictions on use, duration, region. |
| Editorial-Only | No commercial | Per license | No | No | Typically news/documentary context only. |
| Fair Use | Fact-specific | N/A | N/A | N/A | Legal defense, not a license; requires legal review. |
| Unknown / Unclear | Do not use without clearance | - | - | - | Default classification. |
